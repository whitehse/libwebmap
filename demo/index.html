Fiber design

Display: demo/display/ (style + symbols + magnifier + path trace)

  • Cables / drops ≥ z10/z12 · taps & splices ≥ z13
  • Taps: circle + drop-port digit · splices: hexagon
  • Click cable/drop → optical path highlight (needs path_index)
  • Magnifier: cables enter from real map directions (N/E/S/W)
  • Click fiber chip in glass → path-trace that strand
  • Tap through-fiber: warm IN→PT path (≠ fuse ×); drops in red
  • Path list: light from start · hop budget + aggregate loss
  • Double-click glass → full HTML diagram when installed
